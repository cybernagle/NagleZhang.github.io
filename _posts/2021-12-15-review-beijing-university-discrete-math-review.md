重要的推理定律：  
1. ForA x A(x)v ForA B(x) => ForA (A(x) v B(x))  
2. ThereE x (A(x) ^ B(x)) => ThereE x A(x) ^ ThereE x B(x)  
3. ForA (A(x) -> B(x)) => ForA x A(x) -> ForA x B(x)
4. ForA x (A(x) -> B(x) ) => ThereE x A(x) -> ThereE x B(x)
